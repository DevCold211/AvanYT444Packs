How to Install WarCraft

 

Latest Ver 1.4

 

1. Download TechnicLauncher Pick Your System

2. When it's done installing, login to your minecraft account

3. Download 64 bit java if you have it already than go on to the next step

4. Go On Discover and click on launcher settings & go to java and put at least 3.5 or 4 GB

5. Click off it and go to modpacks and search up "WarCRAFT Official"

6. Click install and when done click on play
